# To-Do List Application

## Overview

This project was developed as part of my Web Development Internship (Task 3). It is an interactive To-Do List application built with vanilla JavaScript, focusing on DOM manipulation, event handling, state management, and local data persistence.

## Features

* Create, update, and delete tasks (CRUD)
* Mark tasks as completed
* Filter tasks by All, Active, and Completed
* Persistent storage using `window.localStorage`
* Dynamic DOM updates
* Event delegation for efficient event handling
* Simple and responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

* `index.html` – Application structure
* `styles.css` – Styling
* `script.js` – Application logic

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
